Available Placeholders (exact match required):

  • {{owner_name}} - Name of the stallholder
  • {{business_name}} - Trading name
  • {{booking_id}} - e.g. ESF26-FOOD-0042
  • {{cancel_link}} - Auto-generated secure cancellation URL
  • {{cost}} - Stall price (Only for Confirmed Chargeable)
  • {{bank_details}} - Bank account info (Only for Confirmed Chargeable)
  • {{location_id}} - e.g. Zone A - 01 (Only for Confirmed Free / Location Updates)
  • {{payment_link}} - Stripe Checkout URL (Only for Payment Requested)
  • {{payment_reference}} - e.g. ESF26-FOOD-0042, the booking ID used as the bank-transfer reference (Only for Payment Requested)
  • {{bank_account_name}} - Bank transfer account name (Only for Payment Requested)
  • {{bank_sort_code}} - Bank transfer sort code (Only for Payment Requested)
  • {{bank_account_number}} - Bank transfer account number (Only for Payment Requested)
  • {{reason}} - Rejection reason (Only for Rejected)
  • {{trader_list}} - Formatted list of traders (Only for HCC Batch Check)
  • {{email}} - Trader email address (HCC emails)
  • {{phone}} - Trader phone number (HCC emails)
  • {{registered_business_name}} - Official registered business name (HCC emails)

Note:

The HCC Batch Check template is used when sending multiple trader details to Hull City Council from the HCC Checks page. The {{trader_list}} placeholder will be replaced with a formatted table of all selected traders.
